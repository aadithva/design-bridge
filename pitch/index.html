Architecture
Under the hood
01
Figma Extraction
Parse design files via REST API. Extract design tokens (colors, spacing, typography), detect Plan-of-Record pages, enumerate component variants, and extract text content from frames.
Figma REST API Design Token Extractor POR Page Detector Frame Exporter
02
PR Diff Analysis
Fetch pull request metadata and unified diffs from Azure DevOps. Parse changed files, extract style tokens (padding, colors, border-radius) from code using regex-based analysis.
Azure DevOps API Unified Diff Parser Style Token Extractor
03
Fuzzy Matching Engine
Match Figma files to ADO repositories and pull requests using content similarity, path matching, and naming heuristics. Calculate confidence scores for each potential match.
Content Matcher Confidence Scoring Fuzzy String Matching
04
AI-Powered Comparison
Component-by-component analysis comparing Figma specs against implementation. Checks spacing, typography, colors, border-radius, layout structure, and variant completeness.
Component Matcher Completeness Checker Analysis Engine
05
Report Generation
Produce structured findings with severity levels (error, warning, info, pass). Generate executive summary, component-by-component review, and actionable recommendations.
Severity Classification AI Summary Markdown Export
